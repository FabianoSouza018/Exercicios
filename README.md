# 📚 Exercícios em Linguagem C

## 📌 Descrição

Este repositório contém uma coleção de exercícios desenvolvidos na linguagem C, com foco no aprendizado prático de lógica de programação e fundamentos essenciais da computação.

Os exercícios foram criados como parte de um processo contínuo de estudo, demonstrando evolução desde conceitos básicos até estruturas mais organizadas.

---

## 🎯 Objetivo

* Desenvolver raciocínio lógico
* Praticar entrada e saída de dados (`scanf` / `printf`)
* Trabalhar com operações matemáticas
* Compreender tipos de dados (`int`, `float`, `double`, `char`)
* Aprender estruturas de decisão e controle

---

## 🧠 Conteúdos abordados

### 🔹 Fundamentos

* Declaração de variáveis
* Tipos de dados
* Operações matemáticas

### 🔹 Entrada e saída de dados

* Uso de `scanf`
* Uso de `printf`
* Formatação de saída

### 🔹 Estruturas de decisão

* `if`, `else`
* `switch`

### 🔹 Cálculos matemáticos

* Área de figuras (triângulo, etc.)
* Volume
* Operações básicas

---

## 📂 Lista de Exercícios

Os exercícios estão organizados por arquivos `.c`, representando diferentes problemas resolvidos.

### 🟢 Exercícios iniciais

* **01-exercicio.c** → Cálculo de área com valores fixos
* **002-exercicio.c** → Variação do cálculo anterior
* **02-exercicio.c** → Uso de ponto flutuante (`float`)
* **02-exercicio-area-triangulo.c** → Entrada de dados do usuário
* **03-exercicio.c** → Cálculo de volume

### 🟡 Exercícios intermediários

* Programas com múltiplas entradas
* Uso de variáveis diferentes
* Organização de entrada → processamento → saída

### 🔵 Exercícios em evolução

* Estruturas de decisão (`switch`)
* Validação básica de dados
* Aplicação prática de lógica
