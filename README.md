# 📚 Exercícios em Linguagem C

Repositório com exercícios práticos em linguagem C focado no desenvolvimento de lógica de programação, estruturas básicas e evolução progressiva de habilidades em algoritmos.

---

## 🎯 Objetivo

Este repositório documenta minha evolução prática em programação com a linguagem C, desde conceitos básicos até estruturas mais organizadas de lógica e entrada/saída de dados.

O foco principal é desenvolver habilidades em:

- Raciocínio lógico
- Estruturação de algoritmos
- Manipulação de dados
- Resolução de problemas computacionais
- Evolução contínua na programação

---

## 🧠 Conteúdos abordados

### 🔹 Fundamentos da programação
- Declaração de variáveis
- Tipos de dados (`int`, `float`, `double`, `char`)
- Operações matemáticas básicas

### 🔹 Entrada e saída de dados
- Uso de `scanf`
- Uso de `printf`
- Formatação de saída

### 🔹 Estruturas de decisão
- `if`
- `else`
- `switch`

### 🔹 Cálculos matemáticos
- Área de figuras geométricas
- Volume
- Operações matemáticas aplicadas

---

## 📂 Exercícios

Os exercícios estão organizados em arquivos `.c`, cada um representando um problema resolvido.

### 🟢 Nível iniciante
- `01-exercicio.c` → Cálculo de área com valores fixos  
- `02-exercicio.c` → Uso de ponto flutuante (`float`)  
- `02-exercicio-area-triangulo.c` → Entrada de dados do usuário  
- `03-exercicio.c` → Cálculo de volume  

### 🟡 Nível intermediário
- Programas com múltiplas entradas
- Uso de diferentes tipos de variáveis
- Estrutura entrada → processamento → saída

### 🔵 Evolução contínua
- Aplicação de estruturas de decisão
- Validação de dados simples
- Organização melhor do código
